Spanish Tweets Sentiment Analysis
==============================

An application of traditional and deep learning models to classify Spanish Tweets as positive or negative in sentiment. Explores two techniques of feature extraction using Term frequency Inverse document frequency weighting and Word2Vec word embeddings. Presentation slides available on prezi: https://prezi.com/view/jh8H32IqSm707h33i3vz/
